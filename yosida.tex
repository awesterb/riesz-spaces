\begin{psec}{3.1}{Definitions}
We call an element $x$ 
of a Riesz space $E$
\keyword{infinitesimal}
if the set
\begin{equation*}
\{ n x\colon n\in \mathbb Z \}
\end{equation*}
has an upper bound in $E$.
A Riesz space~$E$ is said to be
\keyword{Archimedean}
if~$0$ is the only infinitesimal element of~$E$.
\end{psec}
